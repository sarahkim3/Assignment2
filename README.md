# Assignment #2 Repository

This repository includes the simulated data for Assignment #2. Fork this repository and add your analysis as described in the canvas assignment.

The csv file for `cohort` in the `raw-data` folder includes 5,000 observations with variables `smoke`, `female`, `age`, `cardiac`, and `cost`.


Summary:
The sample in the cohort.csv file ranges between 18 to 70 years of age and consists of 16% smokers, 55% females, and 22% patients with heart problems. 
Smoking status, age, and cardiac-disease status are positively correlated with healthcare costs, whereas being a female is associated with reduced costs. Accounting for smoking status, gender, and cardiac disease status, a person's age is associated with an average of $15.78 higher cost for every additional year of age. Across all ages, females tend to have lower costs than males, but the relationship between age and cost seem to be parallel between males and females.
